# ふぐカードメーカー（HTML/CSS/JS版）

このフォルダを開いて `index.html` を起動すると、テンプレ画像にテキストを合成したカードPNGを作れます。

## 使い方（いちばん簡単）
- `index.html` をダブルクリックで開く
- 左のフォームに入力 → 右のプレビューが更新
- 「PNGを保存」で画像を保存

> もし「保存ができない / 真っ白になる」などがあれば、下の「ローカルサーバー起動」を使ってください。
> （ブラウザのセキュリティ仕様で、file:// 直開きのCanvas保存が制限される場合があります）

## ローカルサーバー起動（推奨）
Pythonが入っているなら、フォルダで次を実行:
```powershell
python -m http.server 8000
```
ブラウザで開く:
- http://127.0.0.1:8000

## 入力項目
- Playerカード: 名前/サブタイトル/レア度/ふぐ指数/能力/デバフ/豆知識/所属/属性
- Itemカード: 名前/能力/豆知識（デバフ・ふぐ指数は非表示）

## 追加の調整（任意）
- **文字サイズ調整**: 「文字サイズ調整」を開いて、各項目の基準サイズを変更できます（はみ出す場合は自動で小さくなります）。
- **位置調整**: 「位置調整」で、**レア度/ふぐ指数/所属**のブロックを上下に移動できます。
- **画像の切り取り範囲**: 「トリミング調整（ズーム / 位置）」で、ズームとX/Y位置を調整できます。
  - 画像があるときは、**プレビュー上でドラッグ**して位置調整もできます。

## レアリティ別「ふぐ指数」上限
普通 1,000
ちょいレア 2,500
レア 5,000
めっちゃレア 8,500
ウルトラレア 12,000
ばりすごいレア 16,000
ダイナミックレア 20,000

## テンプレ画像
このプロジェクトには、ユーザー提供のテンプレPNGが同梱されています。
